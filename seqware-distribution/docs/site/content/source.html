--- title: Developers Guide --- Developers Guide ================ SeqWare is a Java-based, open-source project using Java 1.6+, Hibernate, Spring, PostgreSQL, Hadoop, Map/Reduce and a variety of other technologies. Developing for SeqWare ---------------------- Please see the plethora of documentation at our [old wiki](http://seqware.sf.net) and feel free to contact the mailing list with any problems you may have. Downloads -------------------- Please see the [Downloads](/downloads/) page for access to source code, SeqWare virtual machines and AMIs.
